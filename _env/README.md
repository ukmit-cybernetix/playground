### Setup Git & Github

Following the steps below:

1. Set up git with your user name and email
2. Open a terminal/shell and type:

   ```
    $ git config --global user.name "Your name here"
    $ git config --global user.email "your_email@example.com"
   ```

3. Check your config:

   ```
     $ git config --list
   ```
